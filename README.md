# Randevo

Randevo, işletmelerin müşteri ve randevu süreçlerini yönetebilmesi için geliştirilmiş bir web uygulamasıdır. ASP.NET Core MVC ve SQL Server teknolojileri kullanılarak geliştirilmiştir.

## Özellikler

* Kullanıcı kayıt ve giriş sistemi
* Rol bazlı yetkilendirme
* Admin paneli
* Müşteri yönetimi
* Hizmet yönetimi
* Randevu oluşturma ve takip sistemi
* Session tabanlı kullanıcı yönetimi
* Güvenli giriş ve yetkilendirme işlemleri

## Kullanılan Teknolojiler

* C#
* ASP.NET Core MVC
* Entity Framework Core
* SQL Server
* Bootstrap

## Proje Durumu

Proje aktif olarak geliştirilmektedir. Yeni özellikler eklenmekte ve mevcut sistemler iyileştirilmektedir.
